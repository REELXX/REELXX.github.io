  3. 这时会在C:\Users\Administrator目录下生成.gitconfig配置文件，我是在C:\Users\94399目录下生成的，还有.ssh文件

    gitconfig会是你刚刚写入的用户配置信息

  4. 生成SSH公钥和私钥


    出现后 按三下回车键
